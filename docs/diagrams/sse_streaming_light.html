事件状态机

  • • 块按 index 严格串行，delta 不交替
  • • contentBlocks[index] 累积每块状态
  • • delta 类型不匹配则 logEvent + throw
  • • raw stream (非 SDK 封装)，避免 O(n²) 解析

关键产出 ★

  • • content_block_stop 每块 yield 一条消息
  • • 即流式工具执行的引信 (addTool)
  • • message_delta 原地 mutate 回填 usage
  • • 另发 raw stream_event 供 UI / TTFT

健壮性

  • • idle 看门狗 90s 无 chunk → abort
  • • stall > 30s 记 tengu_streaming_stall
  • • 无 message_start / 空块 → 非流式降级
  • • max_tokens / context 超限 → 恢复路径
